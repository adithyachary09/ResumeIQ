### ✨ Key Features
* **Deep-Semantic Analysis:** Unlike simple keyword checkers, ResumeIQ maps your experience against JD requirements using advanced LLM inference.
* **Pydantic Enforcement:** Returns perfectly structured, type-safe JSON for seamless frontend integration.
* **Robust Extraction:** Automated PDF-to-text pipeline with built-in error handling for complex document layouts.
* **Health Check:** Integrated monitoring endpoint to verify API and deployment status.
